# 椅子の総数

難易度:★

## 問題
あるイベントに対して準備する机の数と、机１つあたりに必要な椅子の数が与えられたとき、必要な椅子の総数を計算するプログラムを作成してください。

### 入力
入力は以下の形式で与えられる。
```
d c
```

入力は１行であり、必要な机の数d(1≦d≦100)と机１つあたりに必要な椅子の数c(1≦c≦10)を表す整
数が与えられる。

### 出力
イベントにおいて必要な椅子の総数を１行に出力する。
### 入力例1
```
20 2
```


### 出力例1
```
40
```

### 入力例2
```
1 1
```


### 出力例2
```
1
```
