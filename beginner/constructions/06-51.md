# パターンの回転

難易度：★★

## 問題

8文字×8行のパターンを右回りに90度、180度、270度回転させて出力し終了するプログラムを作成して
ください。

### 入力

8文字×8行からなるパターン（半角英数字および半角シャープ“#”と半角アスタリスク“*”）

### 出力

90（半角数字固定）
90度回転させたパターン
180（半角数字固定）
180度回転させたパターン
270（半角数字固定）
270度回転させたパターン 

### 入力例

```
#*******

#*******
######## 
```
