# ２の累乗 
難易度：★★

## 問題
与えられた数を、それ以下の数の中で最大の２の累乗に変換してください。

たとえば、2 や 3 ならば 2<sup>1</sup>=2に変換してください。同様に 4,5,6,7 ならば 2<sup>2</sup>=4 に、8,9,10,11,…,15 ならば 2<sup>3</sup>=8 に変換してください。

### 入力
入力は以下の形式で与えられます。  
１行目：数N(2≦N≦10<sup>6</sup>)

### 出力
変換した数を１行に出力する。

### 入力例１ 
```
54 
```

### 出力例１
```
32
```

### 入力例２ 
```
1024 
```

### 出力例２
```
1024 
```
